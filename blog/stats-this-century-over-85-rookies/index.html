今世紀 プレーオフで85得点以上とったルーキー選手一覧

May 22, 2024 - nba

StatMuseのXにて, 「今世紀にルーキーシーズンのプレーオフで85得点以上取得した選手はManu GinobiliとDereck Livelyのみ」という統計情報が発表されました。

確かにDereck Livelyの活躍はすごかったが, 本当にそれだけか？Dwyane Wadeもルーキーシーズンからプレーオフに行って活躍していた記憶があったので, この統計は疑わしく思い実際に調べてみました。

ルーキーシーズンは, (選手, シーズン)でグルーピングし, その最小のシーズンをとることで取得できます。 ただし, フォーマット上レギュラーシーズンは2から始まり, プレーオフは4から始まるためプレーオフの統計を取得したい場合は先頭の2を4に置き換えて結合する必要がありました。

WITH
  pts_by_season AS (
  SELECT
    PLAYER_ID AS player_id,
    SEASON_ID AS season_id,
    MAX(PLAYER_NAME) AS player_name,
    SUM(PTS) AS total,
    SUM(CASE WHEN MIN > 0 THEN 1 ELSE 0 END) AS game
  FROM
    `nba.leaguegamelog2_player`
  WHERE
    _SEASON_TYPE = "Playoffs"
    AND SEASON_ID >= "42000"
  GROUP BY
    PLAYER_ID,
    SEASON_ID ),
  rookie_season AS (
  SELECT
    PLAYER_ID AS player_id,
    MIN(SEASON_ID) AS season_id
  FROM
    `nba.leaguegamelog2_player`
  WHERE
    _SEASON_TYPE = "Regular Season"
  GROUP BY
    PLAYER_ID )
SELECT
  CONCAT(SUBSTR(p.season_id, 2), '-', LPAD(CAST(CAST(SUBSTR(p.season_id, -2) AS INT64) + 1 AS STRING), 2, '0')) AS season,
  p.player_name, p.total, p.game
FROM
  rookie_season AS r
LEFT JOIN
  pts_by_season AS p
ON
  r.player_id = p.player_id AND
  CONCAT('4', SUBSTR(r.season_id, 2)) = p.season_id
WHERE
  p.total >= 85 -- p.total IS NOT NULL の意も含んでいる
ORDER BY
  p.season_id

実際に検証したところ, 案の定2人だけではなくかなりいることが判明しました。 Dwayne WadeやTyler Herroは確かにルーキーシーズンで活躍した記憶があり, 最初に感じた違和感を晴らしてくれました。 Basketball Referenceも確認したところ, 確かにルーキーシーズンで下記の通りの活躍をしていました。

このクエリで注意すべきこととしては, 「本当の」ルーキーシーズンが判定出来なかった点です。 Chet Holmgrenは2022-23シーズンがルーキーシーズンですが, 1試合もプレー出来ていません。 1試合もプレー出来ていない場合は, 初めてプレーをしたシーズンがルーキーシーズンと見なされてしまうようです。 今後の取り組みとしては, それを見極められるようなテーブルの作成を行いたいです。
