James Hardenが3-Point成功数でRay Allenを抜き歴代2位へ

November 19, 2024 - nba

James Hardenが3-Point成功数でRay Allenの記録を抜いた。 これにて, Stephen Curryに継ぎJames Hardenが歴代2位の記録保持者となった。

このグラフは, 歴代試合出場時間(min)と3-Point成功数である。 HardenもRayを抜いた際, 出場時間がRayよりもだいぶ短かったが, やはり一位のCurryが圧倒的であまり話題にならなかった。

SELECT
  MAX(PLAYER_NAME) AS player_name,
  SUM(FG3M) AS total_fg3m,
  SUM(CASE
      WHEN MIN=0 OR MIN IS NULL THEN 0
      ELSE 1
  END
    ) AS total_games_played,
  SUM(MIN) AS total_minutes_played,
FROM
  `nba.leaguegamelog2_player`
WHERE
  _SEASON_TYPE = "Regular Season"
GROUP BY
  PLAYER_ID
ORDER BY
  SUM(FG3M) DESC
LIMIT 20
